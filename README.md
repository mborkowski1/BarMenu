# Bar Menu

A simple web app built in react.js and node.js. It shows menu items depending on selected menu category.

## Technology Stack

- JavaScript
- React.js
- Node.js
- Express.js
- HTML5
- CSS3
- Bootstrap 4
- NeDB

## App run

In the project directory:

```
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
